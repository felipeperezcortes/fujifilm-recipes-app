# Fujifilm Recipes App (CLI en Python)

Aplicación de línea de comandos desarrollada en Python que guía al usuario en la creación de una configuración (“recipe”) completa para cámaras Fujifilm mediante un flujo interactivo por consola.

El programa permite seleccionar distintos parámetros de cámara y, al finalizar, muestra un resumen ordenado con todos los valores elegidos.

## Objetivo del proyecto
Practicar fundamentos de Python en un contexto aplicado, trabajando principalmente con:
- Control de flujo
- Validación de entradas de usuario
- Lógica condicional
- Uso de estructuras de datos simples

Este proyecto también apoya mi formación en **QA**, reforzando el pensamiento estructurado, la validación de escenarios y la claridad en flujos interactivos.
